=== Cron Table (crontab) 的用法 == crontab 是Linux下的计划任务。基本写法

0 0 * * * $HOME/job.sh

0 0 * * * 这段代表任务执行的时间，分别是“分 时 日 月 周”，指定应的数字即可。

支持逗号和连字符指定多个时间点。如 1-5，1,4,5。还可指定某个区间内每隔一段时间即执行， 1-10/2和1,3,5,7,9是等价的。

# for mass replace

# find, search, and replace
function fsr {
    if [ -z $1 ]; then
        exit_message="filename required"
        echo $exit_message
        exit 1
    fi

    if [ -z $2 ]; then
        exit_message="oldstring pattern required"
        echo $exit_message
        exit 2
    fi

    if [ -z $3 ]; then
        exit_message="newstring required"
        echo $exit_message
        exit 3
    fi

    sed --in-place=bak "s;$2;$3;g" `grep $2 -rl $1`

}
