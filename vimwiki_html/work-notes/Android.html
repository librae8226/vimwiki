    • Setup Build Environment (java)

    if haven't installed yet,
    sudo update-alternatives --install /usr/bin/java java /home/yliao/workspace/toolchains/marvell/bg2/jdk1.5.0_22/bin/java 300
    sudo update-alternatives --install /usr/bin/javac javac /home/yliao/workspace/toolchains/marvell/bg2/jdk1.5.0_22/bin/javac 300

    then, we can use these to change java version:
    sudo update-alternatives --config java
    sudo update-alternatives --config javac

    there will create a link like:
    /usr/bin/java -> /etc/alternatives/java /home/yliao/workspace/toolchains/marvell/bg2/jdk1.5.0_22/bin/java
    OR
    /usr/bin/java -> /etc/alternatives/java/usr/lib/jvm/java-6-sun/jre/bin/java
