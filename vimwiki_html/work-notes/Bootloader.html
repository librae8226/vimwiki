Burn U-Boot

MV88DE3100|> spinit; erase f0000000 f00fffff; tftpboot 0xf0000000 $PATH_TO_UBOOT

NOTE! Step above is legacy. Refer to Yongsen's trac.
http://10.38.54.84/trac/Baseline/wiki/uboot

Burn Specific Image(s)

cd $(TREE)/MRVL/MV88DE3100_SDK/MV88DE3100_Tools
make help
make u2nandimg u2nandimglst='bootimgs'

u2nandimglst can be: block0 bootloader bootimgs kernel boot recovery system cache userdata sysconf
Image is in $(TREE)/MRVL/Release_image/u2nandimg/

Burn Specific Image(s)

img2sd cc/bootloader.subimg b1
img2sd cc/bootimgs.subimg 1
