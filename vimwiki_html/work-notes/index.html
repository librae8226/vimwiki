What Is This?

Here is Librae's rough site, based on vimwiki, see vimwiki - google code or vim.org.
Refer to this page for vimwiki simple guide
一切从简，侧边栏什么就不要了。如果你迷路了，可以看看站点地图。

Any Special?

  1. 本页面完全通过Vim手写，配色也是全凭自己感觉输的数字。
  2. 用到很多 rgba 和 -webkit-gradient ，所以用 IE 的同学悲剧啦！
  3. 底部火焰一样的红色边框，充分展示了 border-image 的强大。
  4. 如果你运气好，应该能看到右上角的自定义Google搜索，马上试一试吧！
  5. 我觉得那个Table of Contents还是很漂亮的哈（快捷键 Shift + Z 可以显示和隐藏哦）。
  6. 集成DISQUS评论服务，有啥话请尽管向楼主说。
  7. 毫无疑问，这个页面在Webkit内核的浏览器里显示效果是最好的（快来试一试[http://www.google.com/chrome/?hl=zh-CN Chrome]或[http://www.apple.com.cn/safari/ Safari]）。

各种注意

  • 本页面支持评论，基于[http://diqus.com DISQUS]实现。但DISQUS有时候实在慢得可以，评论输入框里居然还有iframe。各位如果恰巧不幸赶上DISQUS抽风，又正好想留言的话，还望多一点耐心。
  • 右上角搜索的结果还不完全。这是Google对本站索引太慢造成的。如果你希望有更强力的搜索，就赶快给俺加个友链吧！
